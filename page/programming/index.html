Programming

I’ve been a professional programmer since 2014, I have worked with numerous web technologies, starting from PHP then moving to Python / NodeJS on the back-end, and Angular / ReactJS on the front-end. Today I work mainly as a systems programmer using C/C++, and as a mobile developer using Kotlin/Swift.

You can find a list of my programming projects below:

  • porter: Asynchronous HTTP client for C++(17) using libcurl.
  • gattlib: A fork of gattlib (BLE communication library) with fixes and improvements.
  • q1compile: A GUI to work with Quake 1 map compiling tools.
  • yo: A scripting language for Go.
  • simple_lisp: A simple lisp interpreter.
